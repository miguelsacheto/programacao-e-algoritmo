# 🧠 Programação e Algoritmo

Repositório dedicado aos projetos, atividades e materiais desenvolvidos na disciplina de **Programação e Algoritmo**. Aqui estão reunidos exercícios e trabalhos práticos voltados para o desenvolvimento do raciocínio lógico e da lógica de programação.

## 🎯 Sobre a disciplina

A matéria de Programação e Algoritmo tem como objetivo desenvolver competências relacionadas a:

- Lógica de programação e pensamento algorítmico
- Estruturas de decisão (condicionais)
- Estruturas de repetição (loops)
- Variáveis, tipos de dados e operadores
- Funções e modularização de código
- Estruturas de dados (vetores, listas, etc.)
- Resolução de problemas através de algoritmos

## 🛠️ Tecnologias e linguagens utilizadas

Os projetos deste repositório foram desenvolvidos utilizando diferentes linguagens de programação, de acordo com o conteúdo e os objetivos de cada atividade.
